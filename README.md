# CMS Fullstack Template

Reusable starter template with:
- Express backend
- MongoDB/Mongoose structure
- React frontend
- Shared development notes

## Structure

backend/   -> API server
frontend/  -> React app
docs/      -> development conventions and checklists

## Intended Use
- Copy this project for new apps.
- Add backend modules under `backend/src`.
- Add frontend features under `frontend/src`.
- Keep conventions aligned using `docs/`.
